# 📊 Indian Companies Analysis

This project focuses on **web scraping and data visualization** of the largest companies in India using Python.

---

## 🔍 Project Overview

* Extracted company data from Wikipedia
* Cleaned and processed financial data
* Visualized key insights using different types of plots

---

## 🌐 Data Source

* Wikipedia: *List of largest companies in India*

---

## 🛠️ Technologies Used

* Python
* Pandas
* BeautifulSoup
* Matplotlib

---

## 📈 Visualizations

### 1. Top 10 Companies by Revenue

* Displays the highest revenue-generating companies
* Helps identify market leaders

### 2. Industry Distribution

* Shows number of companies in each industry
* Highlights dominant sectors

### 3. Revenue Distribution (Histogram)

* Shows how company revenues are spread
* Helps understand concentration and variation

### 4. Profit Distribution (Box Plot)

* Displays median, spread, and outliers in profit
* Helps identify highly profitable and loss-making companies

---

## ⚙️ Process

1. Web scraping using `requests` and `BeautifulSoup`
2. Extracting structured data from HTML tables
3. Cleaning data (handling commas, symbols like −, missing values)
4. Converting data types (string → numeric)
5. Creating visualizations using Matplotlib

---

## 💡 Key Learnings

* Handling real-world messy datasets
* Data cleaning and preprocessing
* Understanding distributions and business insights
* Building end-to-end data analysis projects

---

## 🚀 How to Run

1. Install required libraries:

   ```bash
   pip install pandas matplotlib beautifulsoup4 requests
   ```

2. Run the notebook:

   ```bash
   indian_companies_analysis.ipynb
   ```
